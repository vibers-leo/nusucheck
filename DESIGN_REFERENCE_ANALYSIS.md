## 각 서비스별 분석

### 1. 당근마켓 (Daangn Market) - 966개 스크린샷

#### 서비스 개요
- 지역 기반 중고거래 마켓플레이스
- 주변 사람들과의 실시간 거래
- 신뢰도 기반의 커뮤니티

#### 주요 화면/기능
| 화면 | 설명 |
|------|------|
| 홈 | 지역 상품 목록, 카테고리 필터 |
| 상품 상세 | 판매자 정보, 거래방식, 후기 |
| 채팅 | 실시간 구매자-판매자 메시지 |
| 마이페이지 | 내 판매물품, 구매기록, 프로필 관리 |
| 검색 | 상품 검색, 최근 검색 이력 |
| 프로필 | 판매자 평점, 거래 후기, 신뢰 배지 |

#### 디자인 특징

**색상 체계**
```
배경: #1a1b1e (검정), #292a2f (진회색)
텍스트: #ffffff (흰색), #cbcdd2 (밝은회색), #a8abb4 (중간회색)
강조: #b1b0ff (보라색) - 중요한 액션
상태:
  - 성공: #07bc0c (초록)
  - 경고: #e74c3c (빨강)
  - 주의: #f1c40f (노랑)
  - 정보: #3498db (파랑)
```

**타이포그래피**
- 폰트: Pretendard, SUIT, SUIT Variable
- H1: 24px, 무게 600
- H2: 20px, 무게 600
- 본문: 16px, 무게 400
- 보조: 13px, 무게 400
- 라벨: 12px, 무게 400

**공간 시스템 (8px 기반 그리드)**
- 기본 패딩: 16px, 20px, 24px
- 마진: 12px, 20px
- 컴포넌트 갭: 10px, 13px, 17px
- 주요 섹션 간격: 24px

**모양**
- 버튼 & 입력 필드: border-radius 10px
- 카드: border-radius 12-16px
- 프로필/아이콘: border-radius 999px (원형)

#### UX 패턴
1. **하단 탭 네비게이션**: 주요 기능을 아래에 배치 (모바일 친화적)
2. **카드 기반 레이아웃**: 각 상품을 카드로 표현, 일관된 호버 효과
3. **신뢰도 시스템**: 평점, 거래수, 배지로 신뢰성 표시
4. **실시간 알림**: 채팅, 거래 상태 변경 즉시 알림
5. **검색 기록**: 최근 검색어 저장으로 빠른 재검색

## 공통 디자인 패턴

### 1. 색상 아키텍처

모든 앱이 동일한 색상 시스템을 기반으로 함:

**기본 팔레트**
```
Neutral (회색 계열):
  - #1a1b1e: 깊은 배경
  - #262626, #292a2f: 카드/섹션 배경
  - #34363c, #646874: 경계선
  - #a8abb4, #cbcdd2: 텍스트 (보조)

Primary Colors:
  - #b1b0ff: 보라색 (AI, 강조)
  - #3498db: 파란색 (정보)

Status Colors:
  - #07bc0c: 녹색 (성공)
  - #e74c3c: 빨강 (경고/오류)
  - #f1c40f: 노랑 (주의)
```

### 2. 타이포그래피

**폰트 스택**
```css
font-family: Pretendard, SUIT, 'SUIT Variable', sans-serif;
```

**크기 계층**
- H1: 24px, weight 600
- H2: 20px, weight 600
- 본문: 16px, weight 400
- 부제: 18px, weight 400
- 캡션: 13px, weight 400
- 라벨: 12px, weight 400

**라인 높이**
- 제목: 1.4-1.5
- 본문: 1.5-1.6

### 3. 공간 시스템 (스페이싱)

**8px 기반 그리드**
```
2px: 미세한 조정
4px: 컴포넌트 내 간격
8px: 기본 유닛
12px: 작은 요소 간격
16px: 표준 패딩
20px: 섹션 패딩
24px: 주요 섹션 간격
32px: 큰 섹션 간격
```

**컴포넌트 갭**
- 아이콘과 텍스트: 8-10px
- 카드 내부: 12-16px
- 섹션 간: 20-24px

### 4. 형태 (Shape)

**코너 반경**
```
버튼, 입력 필드: 10px
카드: 12px-16px
모달 또는 큰 컨테이너: 12-16px
아바타/프로필: 999px (완전 원형)
```

**그림자**
- 미세한 그림자로 depth 표현
- 다크 배경이므로 밝은 색상 텍스트와 대비

### 5. 반응형 디자인

**중단점 (Breakpoints)**
```
모바일: 0-727px (주력)
태블릿: 728px-1024px
데스크톱: 1025px+
```

**레이아웃**
- 모바일 기반 설계
- 하단 네비게이션 (모바일)
- 상단 네비게이션 (웹)
- 유연한 컨테이너 너비

### 6. 주요 UX 패턴

#### 네비게이션
- **하단 탭 네비게이션** (모바일): 5개 주요 탭
- 현재 탭 하이라이트 (보라색)
- 스크롤 시 숨김/표시 제어

#### 카드 레이아웃
```
구조:
- 상단 이미지 또는 아이콘
- 제목/상품명
- 부제 또는 설명
- 상태 배지
- 하단 액션 또는 메타 정보
```

#### 상태 표시
- 색상으로 즉시 인식
- 배지 또는 라벨로 보강
- 진행률 바 사용

#### 실시간 알림
- 토스트 메시지 (하단 우측)
- 인-앱 알림 패널
- 아이콘 배지 (카운트)

#### 검색 및 필터
- 검색 바 (상단)
- 최근 검색/필터 기록
- 자동완성 지원

## 누수체크 적용 방안

### 1. 색상 팔레트 정의

```css
:root {
  /* 배경 */
  --color-bg-dark: #1a1b1e;
  --color-bg-secondary: #292a2f;
  --color-bg-tertiary: #34363c;

  /* 텍스트 */
  --color-text-primary: #ffffff;
  --color-text-secondary: #cbcdd2;
  --color-text-tertiary: #a8abb4;

  /* 강조 (AI 기술) */
  --color-primary: #b1b0ff;

  /* 상태 */
  --color-success: #07bc0c;    /* 체크 완료 */
  --color-error: #e74c3c;       /* 누수 발견/오류 */
  --color-warning: #f1c40f;     /* 대기/주의 */
  --color-info: #3498db;        /* 진행 중 */
}
```

### 2. 타이포그래피 시스템

```css
/* Tailwind 설정 예시 */
extend: {
  fontSize: {
    'display': ['36px', { lineHeight: '46px', fontWeight: '600' }],
    'h1': ['24px', { lineHeight: '30px', fontWeight: '600' }],
    'h2': ['20px', { lineHeight: '26px', fontWeight: '600' }],
    'body': ['16px', { lineHeight: '24px', fontWeight: '400' }],
    'body-sm': ['14px', { lineHeight: '20px', fontWeight: '400' }],
    'caption': ['13px', { lineHeight: '18px', fontWeight: '400' }],
    'label': ['12px', { lineHeight: '16px', fontWeight: '400' }],
  }
}
```

### 3. 공간 시스템

```css
spacing: {
  'xs': '4px',
  'sm': '8px',
  'md': '12px',
  'lg': '16px',
  'xl': '20px',
  '2xl': '24px',
  '3xl': '32px',
}
```

### 4. 주요 컴포넌트 스타일

#### 버튼
```
패딩: 12px 16px
높이: 48px (터치 최적)
border-radius: 10px
배경: #b1b0ff (주색) 또는 #292a2f (보조)
텍스트: #ffffff 또는 #cbcdd2
호버: opacity 0.8, shadow 추가
```

#### 카드
```
배경: #292a2f
border-radius: 12px
padding: 16px
그림자: 0 4px 12px rgba(0,0,0,0.15)
```

#### 입력 필드
```
배경: #1a1b1e
테두리: 1px #34363c
border-radius: 10px
패딩: 12px 16px
포커스: 테두리 색 #b1b0ff
```

### 5. 누수체크 화면 구조

#### 홈 (고객)
```
홈 화면
├─ 빠른 체크 버튼 (보라색 강조)
├─ 진행 중인 체크 (카드 리스트)
│  ├─ 상태 배지 (파랑)
│  ├─ 주소 및 타입
│  └─ 배정된 마스터
├─ 최근 완료된 체크
└─ 보험청구 현황
```

#### 체크 상세
```
상세 정보
├─ 체크 상태 (색상 코드)
├─ 누수 사진 (AI 분석 결과)
│  └─ 신뢰도 점수 (보라색)
├─ 배정된 마스터
│  └─ 평점 및 후기
├─ 견적서
│  ├─ 작업 내용
│  ├─ 예상 비용
│  └─ 승인 버튼
└─ 채팅 또는 연락처
```

#### 마스터 마이페이지
```
마이페이지
├─ 대기 중인 체크 (노랑)
├─ 진행 중인 체크 (파랑)
├─ 완료된 체크 (초록)
├─ 내 평점 및 평가
└─ 설정
```

   - 제목: "체크가 완료되었습니다"

4. 보험청구 요청
   - 제목: "보험청구서 검토가 필요합니다"
   - 색상: 경고 (#e74c3c)

## 결론 및 권장사항

### 추천 적용 패턴

당근마켓, 미소, 토스의 공통 설계 원칙을 누수체크에 적용하면:

1. **현대적 이미지**: 다크 모드 + 보라색 강조로 기술성 강조
2. **신뢰성 확보**: 명확한 상태 표현과 신뢰도 시스템
3. **사용성 극대화**: 일관된 그리드와 명확한 정보 계층
4. **모바일 중심**: 터치 친화적 디자인과 하단 네비게이션
5. **빠른 인식**: 색상 코딩으로 상태를 즉시 파악

### 차별점

- **당근마켓**: 신뢰도 + 커뮤니티 → 누수체크는 **마스터 신뢰도** + **AI 기술** 강조
- **미소**: 간결성 → 누수체크는 **핵심 기능** (AI 점검, 마스터 배정)에 집중
- **토스**: 금융 안전 → 누수체크는 **투명성** (견적, 에스크로) 강조
